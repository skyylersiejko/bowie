# bowie
EU 2025 Program Unlimited
 Welcome to Stanford! This project is a simple map of part of Stanford University made by Python. It can help you find the shortest way to go from a place to anther. It’s helpful for students, visitors, or any people new to the campus. 

On the map, click your location first(plant a tree).
Then, click the place you want to go, 
It’s going to show you the closest route to your destination.
If want to change, plant another tree and choose another destination.
